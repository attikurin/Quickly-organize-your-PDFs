PDFちょこっと整理（結合・分割・削除）

ブラウザだけで動作する、インストール不要の簡易PDF編集ツールです。
「スキャンしたデータを分割したい」「複数のPDFをまとめたい」「不要なページだけ削除したい」といった日常的なニーズに、安全かつシンプルに応えます。

概要

ファイル名: index.html (または任意の名前.html)

動作環境: Google Chrome, Microsoft Edge, Safari などのモダンブラウザ

通信: 完全オフライン（インターネットへのファイルアップロードは一切行いません）

主な特徴

インストール不要・完全オフライン

HTMLファイル1つで動作するため、ソフトのインストールが制限されているPCでも使用可能です。

すべての処理をブラウザ内（端末内）で行うため、生徒の個人情報やテスト問題、社外秘の資料など、機密データを外部サーバーへ送信することなく安全に加工できます。

直感的な操作

ファイルをドラッグ＆ドロップで追加。

ページの並べ替えはスマホの写真整理のようにドラッグで。

不要なページはゴミ箱アイコンで削除。

シンプル機能

高機能な編集ソフトは重くて使いにくいという声に応え、「結合」「並べ替え」「削除」に機能を絞りました。

使い方

起動する

index.html をダブルクリックしてブラウザで開きます。

ファイルを追加する

画面中央のエリアに、加工したいPDFファイルをドラッグ＆ドロップします。

複数のファイルを一度に追加したり、後から追加（右下の＋ボタン）することも可能です。

編集する

並べ替え: ページのサムネイル（カード）をドラッグして、好きな順番に入れ替えます。

削除: ページ右上のゴミ箱アイコンをクリックすると、そのページが削除されます。

保存する

右上の「結合して保存」ボタンをクリックします。

編集結果が新しいPDFファイルとしてダウンロードフォルダに保存されます。

技術情報

このツールは以下のオープンソースライブラリを使用しており、単一のHTMLファイル内にCDN経由で読み込む形で構成されています。（インターネット接続がない環境で使う場合は、各ライブラリをローカルに保存してパスを書き換える必要があります）

pdf-lib: PDFの作成、ページのコピー・結合処理

PDF.js: PDFの読み込み、サムネイル画像の生成

SortableJS: ドラッグ＆ドロップによる並べ替え機能

Tailwind CSS: ユーザーインターフェースのデザイン

Phosphor Icons: アイコン表示

注意事項

パスワード付きPDF: パスワードで保護されたPDFの読み込みには対応していません（エラーになります）。

ファイルサイズ: ブラウザのメモリを使用するため、極端に大量のページ数や巨大なサイズのPDFを扱うと動作が遅くなる場合があります。数百ページ程度までを想定しています。

上書き保存: 元のファイルを上書きすることはありません。常に新しいファイル名（edited_日付_時刻.pdf）で保存されます。
